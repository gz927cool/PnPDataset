# Index Triplet 深度解析结果

## 项目概述

本目录包含对 PnP (Pattern and Practice) 知识图谱中 **Index Triplet（索引三元组）** 的深度解析结果。
“01-A 深度解析全量结果表 (1-100行).md” 等markdown文件通过人机协作利用大模型直接对原始Index数据原子化爆破解析的结果，解析规则参考 17-Index-LLM规则。
output 目录下存放的是将所有 Markdown 格式的结果汇总并转换为 CSV 格式后的输出文件。

## 目录结构

```
16-Index-Triplet/
├── [01-A ~ 19-UVWXYZ] 深度解析全量结果表 (*.md)   # 按字母分组的解析结果文件
├── output/                                           
│   └── extracted_triplets.csv                        # 转换后的提取三元组（CSV格式）
└── script/                                           
    ├── conver.py                                     # 提取脚本
    └── check.ipynb                                   # 数据校验笔记本
```

## 解析结果文件说明
每个 MD 文件包含三部分：
1. **数据完整性审计报告** - 统计信息
2. **全量原子化解析结果表** - 核心三元组数据
3. **无匹配/已过滤行汇总** - 被过滤的索引项
